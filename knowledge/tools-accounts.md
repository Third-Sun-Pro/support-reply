# Tools & Accounts

## Password Management
- **Dashlane** — Team password manager
- All shared credentials stored in Dashlane
- Team members should have their own Dashlane accounts connected to the team

## Site Tools

### Google ReCaptcha
- Used across client sites for form spam protection
- Third Sun manages the ReCaptcha API keys
- If CAPTCHA isn't working: check API key configuration, check domain whitelist in Google ReCaptcha admin
- Some sites use hCaptcha instead (configured in form extension settings)

### SendGrid
- Email delivery service for transactional emails
- Used when server email is unreliable
- Troy manages SendGrid configuration
- If form emails aren't sending through SendGrid: check SendGrid dashboard for errors, verify API key

### MySites by Joomla
- Security scanner for monitoring Joomla sites
- Troy uses this to check for vulnerabilities and malware
- Can detect outdated extensions, weak passwords, suspicious files

## Font & Stock Photo Accounts

### Font Accounts
- **Adobe Fonts** (included with Adobe Creative Cloud subscriptions)
- **Google Fonts** — free, used on most sites
- Specific premium font licenses may be tied to individual projects

### Stock Photo Accounts
- Team has stock photo subscriptions
- Check with the designer before purchasing new stock photos
- Licensed images should be tracked per project for compliance

## Team Communication & Project Management Tools
- **Missive** — Shared inbox for client support emails
- **Teamwork** — Project management and task tracking
- **Proposify** — Proposal creation and tracking
- **Zapier** — Connects Proposify to Teamwork (automated task creation)
- **Evernote** — Leads tracking and notes
- **Scribe** — Step-by-step client walkthrough guides (scribehow.com)
- **Figma** — Design mockups and collaboration
- **Dropbox** — Shared team file storage

## Adobe Creative Cloud
- **4 Adobe CC subscriptions** (monthly recurring expense)
- Used for: Photoshop, Illustrator, InDesign, After Effects, Adobe Fonts
- Designer is primary user

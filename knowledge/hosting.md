# Hosting Infrastructure

Troy handles all hosting-related issues. Third Sun manages client sites across five providers.

## Providers
- **CloudAccess.net** — Managed Joomla hosting. ~100 sites (largest). Preferred for new sites.
- **Cloudways** — Managed cloud hosting. Used for clients needing dedicated resources (Borchard, UP Colorado, Wasatch Camera Club have their own servers).
- **Stablehost** — cPanel VPS hosting. ~50 sites.
- **Veerotech** — cPanel hosting. ~30 older/legacy sites.
- **XMission** — Shared hosting. Most local nonprofits. Utah-based ISP.

## Backups
- CloudAccess and Cloudways have automated backups with one-click restore
- Stablehost/Veerotech use cPanel backups — Troy manages these
- **Always take a backup before significant changes**

## SSL Certificates
- Route all SSL issues to Troy
- CloudAccess and Cloudways handle SSL automatically (Let's Encrypt)
- Stablehost/Veerotech: Troy manages via cPanel

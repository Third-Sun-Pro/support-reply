# Common Client Tasks

These are the most frequent support requests from clients. All clients use Joomla CMS. When a help doc or Scribe guide exists, link to it rather than explaining the full process.

## Editing Article Content
- Content → Articles → find and click the article → edit in JCE editor → Save
- **Tip:** Use headings (H2, H3) for structure, not just bold text
- **Warning:** Don't paste directly from Word — use "Paste as Plain Text"
- **Guide:** https://thirdsun.com/help-docs/edit-items-and-categories

## Articles & Categories
- Joomla organizes content into **categories** (buckets) and **articles** (items inside them)
- Example: "Staff" is a category, each staff member is an article
- To create: Content → Articles → New → fill in title, content, assign to a category
- Unpublish by toggling the green checkmark — keeps it in the system but hides it from the public site

## Menu Items
- **Joomla is menu-based** — content needs a menu item to show up on the site
- **Warning:** Changing a menu item's alias changes the page URL and can break links
- **If unsure about menu changes, email us** — menu structure can get complicated

## Uploading Photos
- Resize images BEFORE uploading. Minimum 1600px wide, 72 DPI, under 1 MB
- In JCE editor: click image icon → browse server → upload. Always add alt text.
- **Guide:** https://thirdsun.com/help-docs/photos-and-images/size

## Managing Users
- Admin → Users → Manage → create or edit users
- "Administrator" = full backend access. "Registered" = frontend only. "Super User" = reserved for Third Sun.
- To reset a password: find the user → click their name → enter new password → Save

## Homepage
- The homepage is built from **modules**, not a single article
- To edit: find the relevant module in Extensions → Modules, or edit featured articles
- If a client asks about homepage changes, offer to do it for them — homepage structure varies by site

## Duplicating an Article
- Open the article → click "Save as Copy" in the toolbar → edit the copy as needed

## Content Formatting Tips
- Use consistent formatting across articles — don't mix fonts, colors, and sizes
- Left-align body text. Don't over-format (bold/underline everything).
- **H1 is the article title** — don't use it elsewhere. Use H2, H3, H4 in order.
- Shift+Enter = line break (less space). Enter = paragraph break (more space).
- Outside links must include the full URL with `https://`
